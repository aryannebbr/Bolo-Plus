# 🧁 Bolo-Plus - Catálogo de Receitas Gourmet

O **Bolo-Plus** é um projeto Full Stack desenvolvido para gerenciar receitas de confeitaria de forma interativa. A aplicação permite visualizar, buscar e adicionar receitas personalizadas, além de oferecer sugestões externas.

## 🚀 Tecnologias Utilizadas
- **Front-end:** HTML5, CSS3 (Design Responsivo) e JavaScript (Manipulação de DOM).
- **Back-end:** Node.js com Express.
- **Dados:** Persistência em arquivo JSON para simular um banco de dados.
- **API:** Integração com *TheMealDB API* para sugestões aleatórias de sobremesas.

## ✨ Funcionalidades
- **Busca em Tempo Real:** Filtro dinâmico de receitas por nome.
- **Gestão de Receitas:** Adição de novas receitas com validação de dados.
- **Sugestão do Chef:** Botão que consome uma API externa para trazer receitas globais.
- **Interface Premium:** Layout focado na experiência do usuário (UX) com foco em gastronomia.

## 🛠️ Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/aryannebbr/Bolo-Plus.git`
2. Instale as dependências: `npm install`
3. Inicie o servidor: `node servidor.js`
4. Abra o arquivo `index.html` no navegador.
